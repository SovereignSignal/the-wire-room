# The Wire Room

Three beats. One team. Non-stop coverage.

Tracking grants, fellowships, and hackathons across Crypto, AI, and Open Source.

## Tech Stack
- Next.js 16 (Turbopack)
- TailwindCSS
- TeleSum database (PostgreSQL)

## Environment Variables
- `DATABASE_URL` - TeleSum PostgreSQL connection string

## Development
```bash
npm install
npm run dev
```
